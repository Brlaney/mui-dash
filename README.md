# Quick Start

### 1. Clone repository

```C
git clone https://github.com/Brlaney/mui-dash.git
```

### 2. Install dependencies

```C
cd mui-dash

# then for yarn users:
yarn install

# or npm users:
npm i 
```

### 3. Ready for development

```C
yarn dev

#or

npm run dev

```

Now navigate to http://localhost:3000/ and check it out.
